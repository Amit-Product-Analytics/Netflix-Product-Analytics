# Netflix Product Analytics Dashboard

## Project Overview

This project analyzes Netflix subscription, engagement, churn, and content performance data to identify key business insights and support data-driven decision-making.

The dashboard focuses on understanding customer behavior, subscription revenue, retention challenges, content performance, and user engagement patterns.

---

## Business Problem

Netflix needs to answer several critical business questions:

* Which subscription plan generates the most revenue?
* Which customer segments are most likely to churn?
* How engaged are users across different devices and subscription plans?
* Which content genres perform best?
* What business actions can improve retention and revenue?

## Key Insights

### Revenue

* Premium plan generated the highest revenue ($22.5K).

### Churn

* Basic plan recorded the highest churn rate (25%).

### Engagement

* Laptop users showed the highest engagement with an average watch time of 87 minutes.

### Content

* Romance and Sci-Fi achieved the highest completion rate (93%).
